# Деревянная будка - новая структура

## Описание
Добавлена новая редкая структура - деревянная будка, которая спавнится только на полянах (Plains и Sunflower Plains биомы).

## Характеристики:
- **Редкость**: 1 будка на 64 чанка (очень редко)
- **Биомы**: Равнины (Plains) и Подсолнечные равнины (Sunflower Plains)
- **Размер**: 3x3x3 блока
- **Материалы**: Дубовые бревна, доски, стекло

## Содержимое будки:
- Стол для крафта
- Сундук с лутом (хлеб, яблоки, железо, бронза, изумруды, книги, волокно)
- Кровать (случайно)
- Факел для освещения
- Небольшой сад с цветами вокруг

## Технические детали:
- Класс: `WoodenHutFeature`
- Генерация: `GenerationStep.Feature.SURFACE_STRUCTURES`
- Лут-таблица: `caveborn:chests/wooden_hut`

## Файлы:
- `src/main/java/ru/purpir/world/WoodenHutFeature.java` - основная логика генерации
- `src/main/resources/data/caveborn/loot_table/chests/wooden_hut.json` - лут для сундука
- Обновлены файлы конфигурации мира в пакете `ru.purpir.world`
